# Swiggy Power BI Dashboard

## Project Overview
This project is an interactive **Power BI dashboard** created to analyze Swiggy food delivery data.  
The dashboard provides key business insights related to **orders, revenue, customers, restaurants, and delivery trends** using real-world analytical techniques.

---

## Objectives
- Analyze overall order and revenue performance
- Identify top-performing restaurants and food categories
- Understand customer ordering behavior
- Track delivery trends over time
- Apply DAX and Power Query for data transformation and calculations

---

## Tools & Technologies
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query**
- **Excel Dataset**

---

## Dashboard Features
- Total Orders & Revenue KPIs
- Top Restaurants by Orders & Revenue
- Category-wise Sales Analysis
- Time-based Trends (Daily / Monthly)
- Customer & Location Insights
- Interactive Filters and Slicers

---

## Key Insights
- Identified peak ordering times and high-demand food categories
- Found top revenue-generating restaurants
- Observed customer ordering patterns across different time periods
- Helped simulate real-world business decision-making using data
